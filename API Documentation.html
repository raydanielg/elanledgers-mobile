Authentication

Use POST /app/auth/signin to get a token, then pass Authorization: Bearer <token> on protected routes.

Body: application/json or form-data
Request Fields
username Required Phone, email, user ID, or role username.

Banking

  • POST Bearer
    /app/post/postdata/banking/customer-transaction
    Create or update a customer cash-in or cash-out transaction.
    Request Fields
    customer_id Required Customer account holder ID.
    type Required cashin or cashout.
    from_account Required Source account ID.
    to_account Required Destination account ID.
